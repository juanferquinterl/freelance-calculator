# Web Tools

A small collection of simple and useful web tools designed to solve everyday problems quickly.

🌐 Live website:
https://juanferquinterl.github.io/web-tools/

---

## Tools Included

### Freelance Rate Calculator

Calculate your ideal hourly freelance rate based on:

* Desired monthly income
* Monthly expenses
* Hours worked per week

Helps freelancers understand how much they should charge to reach their financial goals.

Open tool:
`/freelance/`

---

### Password Generator

Generate strong and secure passwords instantly.

Features:

* Custom password length
* Uppercase letters
* Numbers
* Symbols

Open tool:
`/passwords/`
